Site Description

This project is an individual assignment for the IT4403 course at Kennesaw State University. The goal is to develop a single-page application (SPA) that allows users to search and display detailed information about books using the Google Books API. The application will include features such as searching for books, browsing by attributes, saving books to a list, and exporting the list in various formats.
